Easy-Guide

This is the graduate project of our college. This system can serve the people that tell them what it is in front of the road. The user can open the app and hold it toward the front. At the same time, the phone would try to connect to the server. The phone would keep catching the image of the front, and send to the server. Next, the server would use image processing algorithm to recognize the distance, direction and type of the obstacle. At last, the result would be sent back and tell the user.

fast-neural-style-tensorflow

This is base on fast-neural-style for my ICG(Interactive Computer Graphics 2017 Fall) term project.

Dazzled-Darling

This is our final project of professor PJ Cheng Web Retrieval and Mining course. We built your own internet celebrity information system from up to 1600 instagram users. Select images you like and query some keywords, we can return 20 instagram users who meet your preference most.

Note: Our system only support Traditional Chinese query. If you are in MacOS, tkinter not support the MacOS orginal Traditional Chinese input, you only can type your query in another place and copy/paste.
